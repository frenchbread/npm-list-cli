# npm-list-cli

> A better `npm list`. Returns a clean list of NPM dependencies.

![Media](https://github.com/frenchbread/npm-list-cli/blob/master/media/Screen%20Shot%202016-09-21%20at%2017.38.43.png?raw=true)

## Usage

```
$ npm-list
```

or

```
$ npm-list --global
```
